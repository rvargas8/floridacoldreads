# Florida B.E.S.T. Standards Alignment

## Overview
ReadReady is designed to align with Florida's Benchmarks for Excellent Student Thinking (B.E.S.T.) Standards for English Language Arts (ELA) in grades 2-5.

## Key B.E.S.T. Standards Addressed

### Reading Comprehension Standards

#### Grade 2
- **ELA.2.R.1.1**: Identify plot structure and describe main story elements in a literary text
- **ELA.2.R.1.2**: Identify and explain a theme of a literary text using text evidence
- **ELA.2.R.1.3**: Identify different characters' perspectives in a literary text
- **ELA.2.R.2.1**: Explain how text features contribute to meaning
- **ELA.2.R.2.2**: Explain the main idea of a text
- **ELA.2.R.2.3**: Explain an author's purpose in an informational text

#### Grade 3
- **ELA.3.R.1.1**: Explain how one or more characters develop throughout the plot in a literary text
- **ELA.3.R.1.2**: Explain a theme and how it develops, using details, in a literary text
- **ELA.3.R.1.3**: Explain different characters' perspectives in a literary text
- **ELA.3.R.2.1**: Explain how text features contribute to meaning and identify the text structures of chronology, comparison, and cause/effect
- **ELA.3.R.2.2**: Identify the central idea and explain how relevant details support that idea in a text
- **ELA.3.R.2.3**: Explain an author's purpose in an informational text

#### Grade 4
- **ELA.4.R.1.1**: Explain how setting, characters, plot, and theme contribute to meaning in a literary text
- **ELA.4.R.1.2**: Explain a stated or implied theme and how it develops, using details, in a literary text
- **ELA.4.R.1.3**: Identify the narrator's point of view and explain the difference between a narrator's point of view and character perspective in a literary text
- **ELA.4.R.2.1**: Explain how text features contribute to meaning and identify the text structures of problem/solution, sequence, and description
- **ELA.4.R.2.2**: Explain how relevant details support the central idea, implied or explicit
- **ELA.4.R.2.3**: Explain an author's perspective toward a topic in an informational text

#### Grade 5
- **ELA.5.R.1.1**: Analyze how setting, characters, and plot contribute to the development of theme in a literary text
- **ELA.5.R.1.2**: Explain the development of stated or implied theme(s) throughout a literary text
- **ELA.5.R.1.3**: Describe how an author develops a character's perspective in a literary text
- **ELA.5.R.2.1**: Explain how text structures and/or features contribute to the overall meaning of texts
- **ELA.5.R.2.2**: Explain how relevant details support the central idea(s), implied or explicit
- **ELA.5.R.2.3**: Analyze multiple accounts of the same event, noting important similarities and differences in the point of view they represent

### Vocabulary Standards
- **ELA.2.V.1.1 - ELA.5.V.1.1**: Use grade-level academic vocabulary appropriately in speaking and writing
- **ELA.2.V.1.2 - ELA.5.V.1.2**: Identify and use base words and affixes to determine the meaning of unfamiliar words in grade-level content

### Reading Foundational Skills
- **ELA.2.F.1.3 - ELA.5.F.1.3**: Use knowledge of grade-level phonics and word-analysis skills to decode words
- **ELA.2.F.1.4 - ELA.5.F.1.4**: Read grade-level texts with accuracy, automaticity, and appropriate prosody or expression

## How ReadReady Aligns

1. **Passage Selection**: All passages are selected to be grade-appropriate and align with B.E.S.T. complexity requirements
2. **Question Design**: Comprehension questions target specific B.E.S.T. benchmarks for each grade level
3. **Text Types**: Passages include both literary and informational texts as required by B.E.S.T. Standards
4. **Vocabulary**: Passages include grade-appropriate vocabulary that supports B.E.S.T. vocabulary standards
5. **Critical Thinking**: Questions require students to analyze, explain, and support answers with text evidence

## Implementation Notes

- Each passage includes questions that require students to cite text evidence
- Questions progress from literal comprehension to inferential and analytical thinking
- Passages are designed to build reading fluency and comprehension skills progressively
- All content is age-appropriate and culturally responsive
